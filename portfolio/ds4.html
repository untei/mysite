作品4
作品名：unison doll ラジオ会館での展示会レイアウト採用案　2004年
担当：展示会立案
ツール：adobe:illustrator
unisonというドールメーカーの展示会のレイアウト立案です。昼の顔と夜の顔ということで、学校の教室ほどのあまり広くないスペースを入口から左回りするようにして昼間の衣装、夜のセクシーっぽさのある衣装、最後に実際にドールに触れてもらえるスペースを設けました。また、どれくらいのポージングを取ることが出来るかということも含めた紹介展示になっています。ドールの数も少ないのでプロの写真家が撮ったパネル写真で雰囲気を味わってもらうことで補いました。

作品名：unison doll 展示会用チラシとコミュニティーサイト運営　2004年
担当：企画・立案・デザイン・制作・運営
ページ作成：html css javascript cgi
ツール：adobe:illustrator photoshop
unisonというドールメーカーの展示会で配布するチラシのデザイン、制作。展示会のコンセプトが昼の顔と夜の顔ということで、コンセプトを活かしたチラシデザイン。文章の基本はメーカー様が作成。販売と同時に希望者にはコミュニティーサイト「ドールタウン」内に無料でレンタルサーバーをプレゼントするという企画を立案、採用。ドールタウンというコンセプトで俯瞰のマップ形式になっており、住人の家（ユーザーのページリンク）が点在、タウン内にいろいろな建物（例えば学校）もあり、その建物に合ったコスプレ写真の投稿と展示、専用掲示板やチャット、写真コンテスト等、ユーザーが楽しめるコンセプトでメーカーとユーザーが一つになったコミュニティーサイトを運営。
