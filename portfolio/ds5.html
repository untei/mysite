作品名：業務webページ lovedoll.jp (dollmate)
担当：デザイン・制作
ツール：adobe:illustrator photoshop dreamweaver
開発：html5 css3 javascript php bootstrap
現在、運営しているラブドールの販売、中古買取を行っているページを制作。ドメインにも表れているようにラブドールを浸透させたのは当社です。2005年から副業という形で運営を開始し、2012年から会社を立ち上げてオリジナルドールの販売も含めて本業としてスタート。買取だけでなく里子里親というシステムでは当社が仲介することで出品者と買取者の個人情報を保護し、中古品をやり取り出来るものです。当社がドールを評価して写真撮影を行い、公開することで安心して購入できることが可能。このシステムは2005年ころ、一部の掲示板で行われていたものをシステム化したものです。

作品名：過去に作成した業務webページの説明　ネスカフェ 2002年作成
担当：制作
ツール：adobe:dreamweaver　Macromedia:fireworks
開発：html css javascript cgi
広告代理店でwebの仕事を担当していた時に、業務で作成したものが唯一、残っていたもの。デザイナーから渡されたものをコーディング。応募フォームにcgiを使用しています。この時代は紙媒体のデザイナーがwebデザインを担当していたため、今だったら背景はテキスタイルで埋め込みをするが、この当時はテキストも含めてベタであった。そのため、一枚画像、そのままでイメージマップの使用が多かったものです。
