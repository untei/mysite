作品8
作品名：過去に運営していたwebページの説明　2003年
担当：企画・立案・デザイン・制作・運営
ツール：adobe:illustrator photoshop dreamweaver
開発：html css javascript cgi
2003年に副業として始めたアダルトグッズショップ。コスプレの中古買取、女性のモニター（今のレビュー）募集等を実施。コスプレはあえて別ドメインとしてコスプレ館とし、運営を分割。アダルトグッズショップからコスプレ館のリンクはありますがコスプレ館からアダルトグッズショップのリンクはありません。当時の風潮としてコスプレ館だけの運営ページもあったため同調した形を選択。可愛らしい名前(Pinky Pocket)とし、女性が利用しやすくするために架空の女性名で運営。仕入れは「トアミ」さんでした。現在は運営しておりません。

コスプレ館(屋号：洋服のクボノヤ)のページです。大半は「トアミ」さんからの仕入れですが一部の商品仕入れは独自にゴスロリを仕入れたり、中古買取品の販売、アメリカからfarisブランドのランジェリーも仕入れて販売しておりました。またゴスロリを充実させてゴスロリ雑誌に広告を打ったりもしました。現在は運営しておりません。
