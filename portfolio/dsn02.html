作品2
作品名：過去に作成した業務webページの説明 ネスカフェ 2002年作成
担当：制作
ツール：adobe:dreamweaver　Macromedia:fireworks
開発：html css javascript cgi
広告代理店でwebの仕事を担当していた時に、業務で作成したものが唯一、残っていたもの。デザイナーから渡されたものをコーディング。応募フォームにcgiを使用しています。この時代は紙媒体のデザイナーがwebデザインを担当していたため、今だったら背景はテキスタイルで埋め込みをするが、この当時はテキストも含めてベタであった。そのため、一枚画像、そのままでイメージマップの使用が多かったものです。
